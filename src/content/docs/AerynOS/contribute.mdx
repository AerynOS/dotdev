## Contributing to our websites (dotcom and dotdev)

AerynOS utilises GitHub to manage this docs site via [this repo](https://github.com/AerynOS/dotdev) and our main site via [this repo](https://github.com/AerynOS/dotcom) The readme of the repo provides the latest guidance on how to build a local version of the site. Our guidance is as follows:

1. Make a localy copy of the repo
2. Create an update branch and build the site locally
3. Make changes as appropriate and ensure those changes do not break the site
4. Once you are ready, submit a PR to the repo.
